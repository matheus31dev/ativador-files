# Ativador UniTV Free - Arquivos de Configuração

Repositório de arquivos de configuração do Ativador UniTV Free.

## Estrutura

- `update.json` - Informações de atualização do ativador
- `ids.json` - Lista de IDs dos arquivos `.config` para ativação

## Como adicionar arquivos .config

1. Coloque seus arquivos `.config` em algum local acessível (Google Drive, GitHub Releases, etc.)
2. Adicione os IDs/URLs ao `ids.json`
3. Faça commit e push
